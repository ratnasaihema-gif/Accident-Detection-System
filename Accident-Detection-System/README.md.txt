# 🚨 Accident Detection & Alert System

## 📌 Overview
This project detects accidents using MPU6050 sensor and triggers an alert.

## 🔧 Components
- Arduino UNO
- MPU6050 Sensor
- Buzzer

## ⚙️ Working
The system monitors acceleration values.  
If the value exceeds a threshold, it detects an accident.
